# CodeAlpha Python Internship

## Overview

This repository contains my completed tasks for the CodeAlpha Python Internship program.

## Tasks Completed

### Task 1: Basic Chatbot

A simple rule-based chatbot built using Python. The chatbot interacts with users by responding to predefined inputs such as greetings, common questions, and basic conversations.

**Features:**

* Responds to user greetings
* Handles common queries
* Easy-to-understand Python implementation
* Beginner-friendly project

### Task 2: Stock Portfolio Tracker

A Python application that helps users track their stock investments. The program calculates the total value of a stock portfolio based on the quantity of shares owned and predefined stock prices.

**Features:**

* Stores stock names and prices
* Accepts user input for stock quantities
* Calculates individual stock values
* Displays total portfolio value
